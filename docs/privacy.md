# 隱私聲明 (Privacy Policy)

**生效日期**:2026-04-27

本站採取**最小蒐集原則**,僅收集匿名統計資料以改善服務,**不收集任何可識別個人之資訊**。

---

## 1. 我們蒐集什麼

### 1.1 自動匿名統計(Cloudflare Web Analytics)

本站使用 Cloudflare Web Analytics 服務蒐集匿名瀏覽統計,內容僅包含:

- 頁面瀏覽量(PV)、不重複訪客數(UV)
- 來源 referrer(您從哪個網站連進來)
- 國家(僅國家層級,不到城市)
- 裝置類型(桌機 / 行動 / 平板)、瀏覽器類型
- 進站與離開頁面、停留時間(粗略)
- HTTP referrer(若有)

**不收集的內容**:
- ❌ 個人身分資訊(姓名、電話、email、IP)
- ❌ Cookie(Cloudflare Web Analytics 完全無 cookie)
- ❌ Local Storage 中之個人偏好(僅暗黑模式狀態,不上傳)
- ❌ 鍵盤輸入、表單內容、搜尋關鍵字(目前不收)
- ❌ 跨站追蹤識別碼(無 fingerprinting)

Cloudflare Web Analytics 之隱私說明:<https://www.cloudflare.com/web-analytics/>(privacy-first 設計,符合 GDPR / CCPA)

### 1.2 使用者主動提供

- **GitHub Issues 留言**:當您於 GitHub 上回報問題或建議,GitHub 會記錄您的帳號名稱、留言內容、時間。本站不另存此資料,完全依 GitHub 隱私政策。
- **Local Storage**(僅存於您的瀏覽器,不上傳):
  - 暗黑模式偏好
  - 條文庫分類樹展開狀態
  - 抽屜開啟過的卡片 ID(用於「最近檢視」未來功能)

---

## 2. 我們不做什麼

- ❌ **不販售**任何資料給第三方
- ❌ **不投放廣告**(本站完全無廣告)
- ❌ **不追蹤**您於本站外的行為
- ❌ **不做使用者畫像**或行為分析
- ❌ **不要求註冊**(無帳號制度)

---

## 3. 為什麼蒐集這些統計

匿名統計用於:
- 了解最常用的母題與情境(優化呈現)
- 找出熱門搜尋關鍵字(補完未涵蓋的條文)
- 評估改版前後的使用情況
- 評估未來推出進階功能(如訂閱、API 服務)的市場可行性

蒐集到的統計**僅作維護者內部分析**,不對外公布原始資料。

---

## 5. Cookie 政策

本站**完全不使用 Cookie**。

- Cloudflare Web Analytics 採無 cookie 設計
- 本站前端僅使用 Local Storage 儲存暗黑模式等偏好,Local Storage 與 Cookie 不同,不會在 HTTP 請求中傳送

---

## 6. 資料保留

- Cloudflare Web Analytics 統計資料保留期間依其服務條款(目前為 6 個月)
- 本站不保留任何使用者層級資料,故無資料刪除需求

---

## 7. 您的權利

由於本站不收集可識別個人之資料,因此 GDPR / 個資法所稱的「查詢權、更正權、刪除權」等對本站不適用。

若您對 Cloudflare Web Analytics 蒐集的匿名統計有疑問,可透過瀏覽器安裝廣告攔截擴充(如 uBlock Origin)阻擋。

---

## 8. 兒童隱私

本站不主動針對 13 歲以下兒童,亦不收集其個資(實際上不收集任何個資)。

---

維護者保留隨時修改本政策之權利。重大修改將更新生效日期並於 GitHub Releases 註記。
