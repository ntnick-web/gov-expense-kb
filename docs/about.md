# 關於本站

## 是什麼

**核銷這樣做!!!**(正式名稱：**政府支出法規知識庫**)是把政府支出相關規定、解釋函令、問答集自動化處理為結構化資料,並用 3 個視圖(核銷情境 / 條文庫 / 試算表)讓使用者快速查找、引用、比較條文的純靜態網站。

> 「核銷這樣做!!!」是站的暱稱,直接告訴同仁「這個情境就照這個方式辦」;正式引用、學術或公文引述時請用全名「政府支出法規知識庫」。

## 目前涵蓋

- **國內旅費** — 13 條報支要點 + 1 份標準表 + 16 份問答集 + 174 份解釋函釋
- **國外旅費** — 24 條報支要點 + 6 份標準表 + 185 份解釋函釋 (BETA)
- **支出憑證與結報** — 24 條處理要點 + 77 份結報問答集 (BETA)

合計 **555 個節點**(550 顯示 + 5 已廢止),皆已建立關聯與標籤索引。資料來源全為政府主管機關(行政院主計總處、外交部等)公開資訊。

## 為什麼做這個

政府法規、函釋、問答集分散在不同 PDF / DOCX 檔案,實務承辦人員常要在多份文件間反覆切換、比對,且彼此引用關係難以追蹤。本知識庫:

- **集中化**:所有條文 + 函釋 + Q&A 一站可查
- **結構化**:每筆都有 ID、標籤、關聯、摘要、原始出處連結
- **視覺化**:核銷情境情境樹 + 條件問答 + 試算表,降低查找門檻
- **保護**:整理內容採 CC BY-NC-ND 4.0 授權,禁商業使用及改作

## 怎麼用

| 使用情境 | 入口 |
|---|---|
| 我有具體核銷情境 | 🧭 **核銷情境** → 選符合的卡片 → 開始條件問答 |
| 我要查特定條文/關鍵字 | 📑 **條文庫** + Ctrl+K 搜尋 |
| 我要試算差旅費或保險費 | 🧮 **試算表** → 輸入城市/天數即知金額 |

每張卡片右上角狀態徽章:
- 🟢 **現行** — 目前有效
- 🟡 **部分修正** — 仍有效但部分條文已修
- ⚫ **已廢止** — 不再適用(預設隱藏,關聯圖仍可見)

每張卡片底部有「🔗 原始出處」連結直連主管機關官方頁面。
