# 關於本站

## 是什麼

**政府支出法規知識庫**是把政府支出相關規定、解釋函令、問答集自動化處理為結構化資料,並用 4 個視圖(母題泡泡 / 核銷情境 / 條文庫 / 關聯圖)讓使用者快速查找、引用、比較條文的純靜態網站。

## 目前涵蓋

- **國內旅費** — 13 條報支要點 + 1 份標準表 + 13 份問答集 + 177 份解釋函釋
- **國外旅費** — 23 條報支要點 + 2 份標準表 + 184 份解釋函釋 (BETA)
- **支出憑證與結報** — 23 條處理要點 + 77 份結報問答集 (BETA)

合計 **513 個節點**,皆已建立關聯與標籤索引。資料來源全為政府主管機關(行政院主計總處等)公開資訊。

## 為什麼做這個

政府法規、函釋、問答集分散在不同 PDF / DOCX 檔案,實務承辦人員常要在多份文件間反覆切換、比對,且彼此引用關係難以追蹤。本知識庫:

- **集中化**:所有條文 + 函釋 + Q&A 一站可查
- **結構化**:每筆都有 ID、標籤、關聯、摘要、原始出處連結
- **視覺化**:核銷情境 + 母題泡泡 + 關聯圖,降低查找門檻
- **公開**:完全開源、CC BY 授權,任何人可自由使用、引用、改造

## 怎麼用

| 使用情境 | 入口 |
|---|---|
| 我大概知道要找哪類規定 | 🫧 **母題泡泡** → 點母題 → 看相關情境 |
| 我有具體核銷情境 | 🎯 **核銷情境** → 選符合的卡片 |
| 我要查特定條文/關鍵字 | 📑 **條文庫** + Ctrl+K 搜尋 |
| 我要研究條文牽連 | 🕸 **關聯圖**(目前隱藏中,程式碼保留) |

每張卡片右上角狀態徽章:
- 🟢 **現行** — 目前有效
- 🟡 **部分修正** — 仍有效但部分條文已修
- ⚫ **已廢止** — 不再適用(預設隱藏,關聯圖仍可見)

每張卡片底部有「🔗 原始出處」連結直連主管機關官方頁面。
