## 2. 視圖一:泡泡概覽圖(P0,首頁)

### 2.1 設計概念

母題視覺地圖,**佔滿整個 view 可用空間**。沒有「中心節點」、不分內外環。

```
┌────────────────────────────────────────────────────────────┐
│   ◯  ●         ●     ◯  ●     ●  ◯                       │
│ ◯       ◯       ◯ ◯           ◯       ◯  ●               │
│   ●         ┌─────────┐                ◯                  │
│  ◯          │ 國外旅費 │       ●  ◯                       │
│             │  208 → 大│              ◯                   │
│       ◯     └─────────┘                                   │
│       ●        ◎     ◎ ◎                                  │
│  ●          ◎    國內旅費   ◎    ●  ◯                     │
│ ◯       ◎ (灰)  ◎    ●  ◎                ●               │
│   ●         ◎     ◎              ◯       ◯               │
│ ◯              ●        ●  ◯                              │
└────────────────────────────────────────────────────────────┘

主泡泡(中央):9 個母題,大小依節點數
裝飾泡泡(外圍):30+ 個,大小不一,半透明,純視覺
```

### 2.2 主泡泡視覺規則

- **9 個母題**:國內旅費 / 國外旅費 / 講座鐘點費 / 酬勞費 / 國外專家 / 其他 / 支出憑證 / 教育部專章 / 國科會專章
- **大小**:依該母題節點數線性映射
  - `r = MIN_R + (count / max_count) × (MAX_R - MIN_R)`
  - `MIN_R = 容器短邊 × 4.5%`,`MAX_R = 容器短邊 × 13%`
- **顏色**:
  - 0 筆 → 灰色(`#95A5A6`),表示「尚未建立檔案」
  - > 0 筆 → 母題色彩(各母題自有色)
- **文字**:只顯示母題名,字級依 `r × 0.30` 動態縮放(12-20px)
  - 不顯示「N 筆」或「無資料」文字
- **互動**:hover scale 1.06,點擊 → 切條文庫並 setFilter(parent)

### 2.3 裝飾泡泡規則

- 數量:依面積比例自動計算(20-60 個,通常 30 上下)
- 半徑:容器短邊 × (1.8% ~ 6.3%) 隨機
- 顏色:9 色循環(同主泡泡色板),`opacity: 0.18 + blur(0.4px)`
- 不可點擊(`pointer-events: none`)、無 label、無 hover 效果

### 2.4 排版演算法

`packBubbles(W, H)` 跑 700 次同步迭代:
- 主泡泡:輕中心引力(`0.003 × (cx - x)`),聚向中央
- 裝飾泡泡:推離中心至 `targetDeco = 短邊 × 45%` 環距,散到邊緣
- 互斥力:`gap = 5`,防重疊
- 邊界限制:`margin = 18`
- 完成後重心校正:依面積加權算重心,平移整個 layout 對齊畫面中心

### 2.5 開場動畫

每個泡泡彈出動畫:`bubble-pop 0.5s cubic-bezier(0.34, 1.56, 0.64, 1)`,間隔 15ms 階遞 delay,內外環依序登場。

### 2.6 圖例

底部小條 banner:「泡泡大小依該母題節點數而定 · 點泡泡進入該母題」

### 3.1 三欄佈局

```

│  核心法規│ ┌─────┐ ┌─────┐         │ ## 條文全文        │

│          │                          │                    │

└──────────┴──────────────────────────┴────────────────────┘

```

### 3.2 分類樹三層結構

```
全部 ─ 母題 ─ 支出類別 ─ 類別代碼
       (9 個)  (依 EXPENSE_LAYER     (核心法規 A /
              從 tag 推斷)            支出標準 B /
                                      解釋函令 C /
                                      問答集 D)
```

「支出類別」中間層由前端 `EXPENSE_LAYER` 表(於 [04_web/assets/app.js](../04_web/assets/app.js))定義。目前已配置:

| 母題 | 支出類別 |
|---|---|
| 國內旅費 | 交通費 / 住宿費 / 雜費 / 程序總則 |
| 國外旅費 | 大陸港澳 / 出國進修 / 交通費 / 生活費 / 手續費 / 保險費 / 行政費 / 禮品交際及雜費 / 程序總則 / 其他 |

「程序總則」是兜底類別,命中泛 tag(結報核銷/總則/法源依據等)而無具體費用 tag 者落入。

### 3.3 鎖定母題模式(從泡泡圖進入時)

- 泡泡圖點某母題 → `setFilter({ parent })` → 條文庫進入「鎖定」
- 樹根直接顯示該母題(藍色加粗,虛線下劃線)
- **不顯示其他母題**(scope 內視野)
- 支出類別預設展開
- 底部「↩ ← 全部母題」按鈕清 scope 回到「全部母題」模式

預設模式(尚未進入鎖定):樹根「全部 27」+ 各母題折疊。

### 3.4 條文卡片

- 每張卡片顯示:ID、標題、摘要(3 行截斷)、tag(前 3 個)
- **草稿 vs 已校對**:
  - 已校對(有 `reviewed`):標題黑色、無 flag
  - 草稿(無 `reviewed`):標題灰色、右上「草稿」橘 flag
- 左邊框依類別著色:A 藍 / B 綠 / C 橘 / D 紫
- 點擊 → 開抽屜
- 卡片群組依類別排序(核心法規 → 支出標準 → 解釋函令 → 問答集),每組顯示「類別 — N 筆」標題

### 3.5 抽屜

- 寬 460px,可收合
- 標頭:ID badge + 標題 + 關閉
- Metadata 行:類別、母題、機關、版本、發文字號(若有)、校對日(若有)
- Body:從 [02_markdown](../02_markdown/) 動態 fetch MD 並用自製簡易 parser 渲染
- **反向連結區塊**:「本節點被以下引用 (N): ...」(從 `incomingEdges` 推導,可點跳轉)
- 底部:複製引用、複製連結

### 3.6 標籤雲

頁面上方,動態反映 visible nodes 內前 20 高頻 tag:
- 點 tag → 過濾(已選 chip 變藍底白字)
- 再點 → 取消過濾

### 3.7 互動

| 操作 | 行為 |
|------|------|
| 點分類樹 | 過濾 + 麵包屑更新 + 標籤雲更新 + 「清除過濾」按鈕出現 |
| 點卡片 | 開抽屜 |
| 抽屜內反向連結 | 跳到對方節點(切視圖+開抽屜) |
| `Ctrl+K` / `/` | 喚起搜尋 |
| `Esc` | 關抽屜 / 關搜尋 |
| `←/→` | 抽屜開啟時切換上下卡片 |
| URL `#A-國內旅費-005` | 啟動時自動開該抽屜 |

## 4. 視圖三:關聯圖(P2)

### 4.1 版面

全螢幕 SVG。節點 = 條文/函釋/Q&A,連線依 `relation` 不同樣式。

### 4.2 靜態 layout(無 RAF 動畫)

- 打開時跑 `runPackingSync(W, H, 320)` 同步收斂(~10-50ms),然後一次 paint
- **不啟動 RAF**:打開即穩定布局,不會看到節點彈跳
- 拖曳節點:mousemove 直接 paint(無模擬),mouseup 釋放 fx/fy 但保留新位置
- 「重新模擬」按鈕:重新隨機散布初始位置 + 跑 320 次同步 packing + paint

### 4.3 邊樣式

| relation | 來源 | dasharray | 額外 |
|---|---|---|---|
| `cites` | 人工 | `none`(實線) | |
| `explains` | 人工 | `6 4`(虛線) | |
| `answers` | 人工 | `2 3`(點線) | |
| `belongs_to` | 人工 | `3 3` | |
| `cites_inferred` | 自動 | (基底)+ `data-inferred="true"` | `stroke-opacity 0.22; stroke-dasharray: 1 5; stroke-width: 1` |
| `answers_inferred` | 自動 | 同上 | 同上 |

推斷邊在前端用更淡的點線顯示,過濾面板「自動推斷」checkbox 可關。

### 4.4 節點樣式

- 顏色依 `type` 區分:A 藍 / B 綠 / C 橘 / D 紫
- 已校對(有 `reviewed`)節點:綠色描邊
- Hover:加深邊 + 一階關聯高亮(8 條邊高亮、無關節點淡化)

### 4.5 Scope 過濾(共用條文庫的 `filter.parent`)

- 從泡泡圖點母題 → 條文庫 setFilter → 切到關聯圖 → 自動套 scope
- 顯示該母題節點 **+ 跨母題鄰居**(任一邊跨到該母題,對方納入)
- 左上 banner:「目前範圍:**XX** [顯示全部]」
- 點「顯示全部」清 `filter.parent`(也影響條文庫,回全部母題模式)

### 4.6 過濾控制(右上面板)

```
過濾
─ 類別 ─
☑ ● 核心法規
☑ ● 支出標準
☑ ● 解釋函令
☑ ● 問答集
─ 關聯 ─
☑ ─── 引用
☑ ┄┄┄ 說明
☑ ⋯⋯⋯ 解答
☑ ⋯⋯⋯ 自動推斷
─ 顯示 ─
☑ 顯示孤立節點
☑ 顯示節點標題
[ 重新模擬 ]
```

### 4.7 互動

| 操作 | 行為 |
|------|------|
| 點節點 | 切條文庫 + setFilter(parent) + 開抽屜 |
| 拖曳節點 | mousemove 跟隨,mouseup 釋放但保留新位置 |
| Hover 節點 | 顯示 tooltip(ID/類別/母題/摘要),一階關聯高亮 |
| 過濾 checkbox | 即時更新顯示 |
| 重新模擬 | 重新散布 + 跑 packing |
| Tab 鍵 | 聚焦節點(`tabindex=0`),Enter/Space 觸發點擊 |

## 5. 全域搜尋

### 5.1 喚起

- 點頂部搜尋框 / `Ctrl+K` / `Cmd+K` / `/`

### 5.2 搜尋實作

純 vanilla JavaScript substring 比對(無 FlexSearch 依賴):
- title 命中 5 分、tags 命中 3 分、body 命中 1 分、summary 命中 1 分
- 結果排序依分數降序,顯示前 12 筆
- 結果含 ID、類別、母題、摘要片段(snippet),關鍵字 `<mark>` 高亮

### 5.3 鍵盤操作

- `↑/↓` 移動焦點
- `Enter` 跳轉(切條文庫 + 開抽屜)
- `Esc` 關閉

### 5.4 進階搜尋語法(P3,未實作)

預留:`tag:交通費` / `type:函釋` / `parent:國內旅費`
