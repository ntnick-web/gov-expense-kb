## 標準全文

中央各機關(含事業機構)派赴國外進修、研究、實習人員補助項目及數額表（中華民國 111 年 11 月 16 日行政院院授主預字第 1110103598 號函修正，並自 112 年 1 月 1 日生效，單位：美元）

### 一、月支生活費

依「日支數額級距」分級：

| 日支數額級距 | 月支生活費 |
|------|------|
| 410 以上 | 1,700 |
| 370–409 | 1,600 |
| 330–369 | 1,500 |
| 290–329 | 1,400 |
| 250–289 | 1,300 |
| 210–249 | 1,200 |
| 170–209 | 1,100 |
| 169 以下 | 1,000 |

備註：

1. 出國期間(依曆計算，以下同)在十五日以下部分，每日按「國外出差旅費報支要點」第七點第一項由行政院另定之「中央政府各機關派赴國外各地區出差人員生活費日支數額表」(以下簡稱日支數額表)全額支給。但有供膳宿、供宿不供膳、供膳不供宿情形者，分別按原支數之一折、三折、八折支給。
2. 出國期間逾十五日部分，參照日支數額表所列城市之日支數額，按左表分級補助，第十六日至第三十日，每日按左表數額二十分之一支給，第三十一日起，每月按左表數額支給；其未滿整月之畸零日數，每日按左表數額三十分之一支給。但有供膳宿、供宿不供膳、供膳不供宿情形者，分別按原支數之三折、四折、九折支給。
3. 前二點所定供宿，包括住宿免費宿舍、過境旅館、在搭乘之交通工具歇夜及返國當日在內。
4. 出國進修、研究、實習人員不得報支「國外出差旅費報支要點」第十五點、第十六點及第十七點所定之行政費(報名費及註冊費除外)、禮品交際及雜費。

### 二、出國手續費及往返機票費

檢據覈實報支。

備註：

1. 出國手續費按「國外出差旅費報支要點」第十三點規定報支。
2. 出國、返國一律搭乘經濟(標準)座(艙)位。

### 三、每學年學雜費（含報名費、註冊費、訪問學人費、實驗費、必要之會費及設施使用費）

由各機關依核准之計畫檢據覈實報支。

### 四、每學年觀摩實習及交通費

由各機關依核准之計畫覈實報支。

備註：

1. 交通費檢據覈實報支。
2. 生活費依日支數額表報支。但應按實際日數核算扣除原支之月支生活費，不得重複支領。

### 五、綜合補助費

每月 180 美元。

備註：

1. 發給內涵包括補助書籍費、綜合保險費、健康保險費、內陸交通費(含租車費)、論文寫作等費用。
2. 按月計發，未滿整月之畸零日數，在十五日以內者，按半個月發給，逾十五日者，按一個月發給。
3. 由當事人簽名或蓋章具領。

### 附記

1. 中央各機關(含事業機構)派赴國外進修、研究、實習人員之補助按本表規定報支。但各主管機關得在本表規定數額範圍內視經費狀況自行核酌支給。
2. 各級地方政府派赴國外進修、研究、實習人員之補助，準用本表規定報支。
3. 本表修正生效後，出國期間跨越新、舊規定者，其於舊規定出國期間適用舊規定，於新規定出國期間適用新規定，但於新規定出國期間適用新規定之補助總額低於適用舊規定者，得適用舊規定。

## 重點摘要

中央各機關（含事業機構）派赴國外進修、研究、實習人員之補助按本表辦理（111-11-16 行政院修正、112-01-01 生效，單位：美元）。月支生活費依日支數額級距分 8 級（169 以下→1,000；410 以上→1,700）；出國手續費及往返機票費覈實報支；每學年學雜費、觀摩實習及交通費依核准計畫報支；綜合補助費每月 180 美元（含書籍、保險、內陸交通、論文寫作等）。在臺日數、供膳宿、跨新舊規定等細節詳見備註與附記。
