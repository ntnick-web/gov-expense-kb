## 標準全文

支標手冊 / 第壹篇  各項支出標準及其規定 / 二、業務費 / (二)稿費 / 相關解釋
(業務費-稿費4)校對費支給疑義類別內容校對費支給疑義
(行政院主計總處104.8.6主預教字第1040101624號「主計長信箱」)
行政院於104年7月15日修訂支給要點，其中校對費支給基準，考量電腦作業進步，實務上均採照相製版，不論撰稿或編稿均已大幅減少校對需要，且已有審查支給項目，應本撙節原則，避免重複支給，惟為應特殊案件之需，仍將原規定之按「稿酬」5%至10%支給，修正為按「撰稿費」之5%至10%支給，爰本案仍請依上開規定，本撙節原則辦理。

## 重點摘要

校對費支給基準，條文規定由原按「稿酬」5%至10%修正為按「撰稿費」5%至10%支給，應本撙節原則避免重複支給。
