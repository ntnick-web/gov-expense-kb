## 調解委員會委員究應支給出席費或兼職費

調解委員會委員究應支給出席費或兼職費(原行政院主計處 95.6.6 處實一字第 0950003506 號函)1.依「鄉鎮市調解條例」第 3 條規定：「調解委員會委員，由鄉(鎮、市)長遴選 鄉(鎮、市)內具有法律或其他專業知識及信望素孚之公正人士，提出加倍人數 後，並將其姓名、學歷及經歷等資料，分別函請管轄地方法院或其分院及地方 法院或其分院檢察署共同審查，遴選符合資格之規定名額，報縣政府備查後聘 任之」。同條例第 7 條規定：「調解委員會調解時，應有調解委員三人以上出席。 但經兩造當事人之同意，得由調解委員一人逕行調解」。2.因調解委員係無法定俸給薪資之榮譽職務，自無兼職費之疑義，是以歷年調解 委員僅於出席調解會議時領取出席交通費，爰調解委員會委員出席調解會議宜 支給出席費。
