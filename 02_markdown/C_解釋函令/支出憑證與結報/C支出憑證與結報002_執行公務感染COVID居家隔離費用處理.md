## 執行公務感染COVID居家隔離費用處理

各機關人員因執行公務感染嚴重特殊傳染性肺炎（以下簡稱COVID-19），或因執行公務與確定病例接觸，接獲衛生主管機關通知應實施居家隔離所衍生相關費用之處理原則如說明（詳附件函）。
